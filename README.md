# TravelTrucks

https://github.com/AlisaPagan/travel-trucks

TravelTrucks is a camper rental web application built with Next.js and TypeScript. It allows users to browse campers, filter them by different criteria, view detailed information about each camper, and submit a booking form.

## Features

- Browse a catalog of campers
- Filter campers by location, form, transmission, and engine type
- Load more campers with pagination
- View detailed camper information
- Image gallery for each camper
- Reviews section
- Booking form

## Tech Stack

- Next.js
- React
- TypeScript
- TanStack Query
- Axios
- Swiper
- CSS Modules
- Loader
